# CALCULATOR #

Live link: https://fshenton.github.io/calculator/

Features:
- Power/clear button
- Add
- Subtract
- Multiply
- Divide
- Square root
- Index 
- Negative numbers
- Decimal places
- Stretch goals: MRC, M- and M+ functions

Written with HTML, CSS and JS.

To run the project:
- Clone the repository
- Open index.html
